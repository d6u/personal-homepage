Patterson Park WordPress Theme

Patterson Park WordPress theme is a class project aimed to improve the information architecture and design of Patterson Park website (http://pattersonpark.com/).

This project is developed with WordPress and PHP. It utilizes responsive layout to provide a better user experience and interface on mobile devices.

Map Project

Map Project is a web application developed with MEAN stack (MongoDB, Express.js, Angular.js and Node.js).

It helps people mark places and display complicated routes between places on a single map at the same time.

D3 Data Visualization

This is a simple data visualization project using D3.js and Angular.js.

The purposes is simple, group Facebook friends by their horoscopes and do some simple analysis with graphics. It triggers animation when switching between different parameters.
